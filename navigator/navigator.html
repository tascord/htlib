Basics

To create a navigator driven page, simply import the CSS & JS:
<link rel="stylesheet" href="navigator.css"> <script src="navigator.js" defer></script>
You're also free to define some basic settings:
<script>  const NavigatorOptions = {   in_time: 750,   out_time: 750,   hang_multiplier: 0.75,  } </script>
Then, simply create some sections!
<section name="main">  <div class="container full">  <button href="/" class="full">Home</button>  <button to="docs" class="full">Docs</button>  </div> </section> <section name="docs">  ... </section>
Note that section named "main" will be loaded on page load, if it exists.

Buttons

You can use buttons to switch between sections by defining a "to" property, which provides a section name:
<button to="main" class="full">Back</button>
You can also use buttons like anchor tags by defining a "href" property like so:
<button href="/" class="full">Home</button>

Javascript Functions

You can switch the current section by calling the NavigateToSection(section, instant?)
function, where section is the name of the section and instant is a boolean which, if true will switch ignoring the in_time & out_time settings.
